Calculadora Simples

Descrição:
Calculadora em Python para terminal, com suporte a operações básicas, números decimais, histórico dos últimos cálculos e comandos especiais como limpar ou apagar o último caractere.


---

Funcionalidades

Operações matemáticas: +, -, *, /

Suporte a números decimais

Comando b → apaga o último caractere do último cálculo

Comando c → limpa a calculadora

Comando sair → encerra o programa

Histórico dos últimos 5 cálculos



---

Como usar

1. Baixe o arquivo calculadora.py


2. Abra no Pydroid ou terminal Python


3. Digite a operação desejada e pressione Enter


4. Use os comandos especiais conforme necessário:

b → apagar último caractere

c → limpar tudo

sair → encerrar




Exemplo de uso:

Digite a operação: 2 + 3 * 4
Resultado: 14
Histórico recente:
2 + 3 * 4 = 14


---

Observações

Digite operações válidas; caso contrário, o programa exibirá “Erro!” e continuará rodando.

O histórico exibe os últimos 5 cálculos, sempre mostrando a expressão e o resultado.
